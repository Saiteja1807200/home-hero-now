# Welcome to this project

## Project info
🏠 Home Hero — Service Booking Platform
A full-stack service marketplace built using a modern Backend-as-a-Service approach.
The application enables users to discover and book home services while leveraging Supabase for authentication, database management, and backend logic.

## 📌 Problem Statement
Users often struggle to find reliable home service providers with transparent pricing and availability.
Most platforms lack structured booking flows and verified user interactions.

## 💡 Solution
Home Hero provides a centralized platform where:
Users can explore available services and providers
Book services with a structured workflow
Track bookings and interactions in real time

## 🚀 Key Features
Secure authentication using Supabase Auth
Service listing and management stored in PostgreSQL
Booking system with user-specific data tracking
Role-based access (user vs service provider)
Real-time updates using Supabase subscriptions
Responsive UI across devices

## 🧱 Tech Stack
Frontend
React.js
Tailwind CSS
Backend / Services
Supabase (Database,Auth,APIs)
Database
PostgreSQL (via Supabase)
## ⚙️ Architecture Highlights
Eliminated traditional backend by using Supabase APIs
Direct client-to-database interaction with secure policies
Row Level Security (RLS) for data protection
Modular frontend structure for maintainability
